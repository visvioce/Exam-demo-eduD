## 🚀 快速启动

### 方式一：后台启动（推荐日常使用）
```bash
./start.sh
```

### 方式二：调试启动（推荐首次使用，查看实时输出）
```bash
./start-debug.sh
```
然后分别在两个终端运行：
1. 终端1：`cd backend && mvn spring-boot:run`
2. 终端2：`cd frontend && npm run dev`

### 访问系统
- 📱 前端地址: http://localhost:5173
- 🔧 后端地址: http://localhost:8080
- 📚 API文档: http://localhost:8080/swagger-ui.html

### 停止系统
```bash
./stop.sh
```

## 🔍 常见问题排查

### 问题1：前端一直加载不出来
**解决方法：**
1. 先使用 `./start-debug.sh` 调试模式
2. 分别在两个终端启动，查看实时输出
3. 检查后端日志：`tail -f logs/backend.log`
4. 检查前端日志：`tail -f logs/frontend.log`

### 问题2：后端启动失败
**可能原因：**
- 数据库连接失败
- 端口8080被占用
- Maven依赖下载失败

### 问题3：前端启动失败
**可能原因：**
- 端口5173被占用
- node_modules损坏（删除后重新npm install）
