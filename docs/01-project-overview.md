# 南方职业学院在线考试系统 - 项目概述

## 一、项目简介

这是一个基于 **前后端分离架构** 的在线考试系统，支持多角色（管理员、教师、学生）使用，提供课程管理、题库管理、试卷管理、在线考试、自动评分、AI智能出题等功能。

### 核心功能

| 功能模块 | 描述 |
|---------|------|
| 用户管理 | 用户注册、登录、角色权限控制 |
| 课程管理 | 课程创建、选课、课程成员管理 |
| 题库管理 | 多种题型（单选/多选/判断/填空/简答）支持 |
| 试卷管理 | 手动组卷、自动组卷 |
| 在线考试 | 实时答题、自动保存、防作弊全屏模式 |
| 自动评分 | 客观题自动评分、主观题教师评分 |
| AI出题 | 集成通义千问、DeepSeek等AI模型智能生成题目 |
| 公告系统 | 系统公告发布与管理 |

## 六、快速开始

### 环境要求

- Java 17+
- Node.js 18+
- MySQL 8.0+
- Maven 3.8+

### 启动步骤

```bash
# 1. 初始化数据库
mysql -u root -p < database/init.sql
mysql -u root -p < database/data.sql

# 2. 启动后端
cd backend
mvn spring-boot:run

# 3. 启动前端
cd frontend
npm install
npm run dev

# 或使用启动脚本
./start.sh
```
